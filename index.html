Web & App Development Workshop

About the Event

This workshop will demonstrate how to quickly & easily you can build web & mobile backend applications using Heroku, part of the Salesforce App Cloud, using a $7 Keepod Android device. Using free, open source technologies such as Java, Php, Ruby, Angular, Bootstrap, Ember, React etc. you can start your own business using Heroku!

  • Date:May 6, 2016
  • Location:UNICEF Innovation Hub, Polytechnic Chichiri Campus, Blantyre
  • Total Ticket: Free to the Public
  • Speaker:David Baliles from Salesforce/Heroku
  • Speaker:Carl Meyer from SourceIT/Keepod
  • Contact:Contact hello@shiftit.co.za for details
More About Event

Schedule & Sessions

  • 6 May

Session 1

9:00am - 9:30am

Welcome & Opening Remarks
Speaker: Carl Meyer - SourceIT

10:00am - 10:30am

What is Heroku
Speaker: David Baliles - Salesforce

In this session David will discuss how hosting technologies have evolved so you no longer need an entire server to host a web or mobile app, but just a 'container' in which to run it. You will learn what Heroku is, how it relates to Salesforce.com and how it can help you create, deploy and manage your web applications easily and cost-effectively.

Session 2

10:30am - 11:00am

Building your First Heroku Application
Speaker: David Baliles

In this session David will demonstrate how to log into Heroku and begin creating a new web application, including creating the container of compute, the additional services required to host the web application, and stitching everything together to have a fully-working application.

11:00am - 11:15am

Break

Session 3

11:15am - 12:00pm

Hands On Workshop
Speaker: David Baliles

In this session, attendees will learn how to sign up & deploy an application on Heroku through a quick, hands-on activity.

12:00pm - 12:30am

Ways to Get Involved
Speaker: Carl Meyer

Carl will conclude our day with a brief overview of SourceIT & introduce opportunities for how you can get involved and begin creating web and mobile applications of your own.
